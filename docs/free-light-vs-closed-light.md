# Free Light vs Closed Light

## Conceptual Split

Perfect Closure distinguishes two regimes of light-like phase behavior:

1. **Free light** - non-closing propagation.
2. **Closed light** - self-returning standing-wave closure.

The same phase ontology appears in both cases. The difference is whether phase closes.

## Free Light

Free light is modeled as null propagation with no stable closure in a rest frame:

\[
P\bar P = 0.
\]

Interpretation:

- propagation remains open;
- phase does not form a persistent closed shell;
- energy is carried in transit rather than stored in a stable rest structure.

## Closed Light

Closed light is light-like phase that self-organizes into a stable return:

\[
P\bar P = m^2 c^2 > 0.
\]

Interpretation:

- phase returns on itself;
- a standing-wave closure is sustained;
- invariant norm appears as mass shell.

## Closure Language

The distinction can be summarized:

\[
\text{free light} \rightarrow \text{open/null/non-closing},
\]
\[
\text{closed light} \rightarrow \text{standing/massive/self-closing}.
\]

This is a structural statement. It does not imply that all microscopic dynamics are solved.

## Why This Matters

The free-vs-closed split gives the minimal ontology needed for the bridge:

\[
\text{phase propagation} \rightarrow \text{phase closure} \rightarrow \text{mass shell}.
\]

Hydrogen provides the first calibrated example where closure shells are mapped to measured spectral structure.
