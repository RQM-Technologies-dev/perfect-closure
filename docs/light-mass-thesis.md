# Light -> Mass Thesis

## Core Statement

Perfect Closure is the thesis that:

> **It is all light; light that has closure becomes mass.**

The claim is structural. "Light" here means propagating phase behavior. "Mass" means a phase configuration that has become self-closing and can persist as a standing resonance.

## Physical Reading

- **Free light:** non-closing null phase propagation.
- **Mass:** light-like phase that self-closes into a stable quaternionic standing-wave resonance.
- **Stable matter:** persistent closure shell.
- **Radiation release:** non-closure channels or de-closure channels.

## Transition Spine

\[
\text{free light}
\longrightarrow
\text{phase closure}
\longrightarrow
\text{mass shell}
\longrightarrow
\text{atomic spectrum}
\]

This repository treats that transition as the main scientific spine.

## Why This Is a Stronger Center

The previous critical-line framing was mathematically suggestive but proof-adjacent in rhetoric. The current framing is cleaner:

1. It starts from a direct ontology (free phase vs closed phase).
2. It maps that ontology into quaternionic geometry.
3. It links geometry to measurable shell spectra (hydrogen bridge).
4. It keeps calibration and falsifiability explicit.

## Boundary Statement

This thesis is a framework and research program, not a finished fundamental theory. It is intended to be rigorous, falsifiable, and incrementally calibratable.
