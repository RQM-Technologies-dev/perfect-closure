\begin{abstract}
This paper provides the calibration and falsifiability layer for the Perfect Closure series. The formal bridge
\[
m_n=\frac{\hbar}{cL_*}|t_n|
\]
becomes predictive only if $L_*$ is independently fixed or stably shared in a predeclared physical family. We define resolved closure triples, calibration regimes, and failure criteria.
\end{abstract}

\section*{Reader Guide}
This paper asks whether the bridge predicts anything testable or merely restates known structures. It sets evidence standards and falsification criteria.

\section{Introduction}
Papers 1--4 provide a formal map from zeta closure to mass-shell form. Here we ask when that map becomes predictive.

\section{Recap of the bridge}
\[
k_n=\frac{t_n}{L_*},\qquad k_C=\frac{mc}{\hbar},\qquad m_n=\frac{\hbar}{cL_*}|t_n|.
\]
The equation is structural unless $L_*$ is constrained independently.

\section{Resolved scale relation}
\[
L_*=t_n\frac{\hbar}{mc}=t_n\bar\lambda_C,
\qquad
\bar\lambda_C=\frac{\hbar}{mc}.
\]
This relation expresses internal-to-physical scale conversion.

\section{Resolved Perfect Closure}
\begin{definition}[Resolved closure triple]
A triple $(t_n,L_*,m_n)$ is resolved if
\begin{itemize}
  \item $\Xi(t_n)=0$,
  \item $\mathcal A_{\mathbf u}\chi_{t_n}=t_n\chi_{t_n}$,
  \item $P\bar P=m_n^2c^2=(\hbar t_n/L_*)^2$.
\end{itemize}
\end{definition}

\begin{remark}
Resolved consistency is necessary but not sufficient for predictive success.
\end{remark}

\section{Calibration regimes}
\begin{enumerate}
  \item Particle-specific $L_*$ (weakest predictive content).
  \item Family-specific $L_*$ (testable if stable).
  \item Universal $L_*$ (strongest and most falsifiable).
\end{enumerate}

\section{Ratio and stability tests}
If one $L_*$ is shared,
\[
\frac{m_a}{m_b}=\frac{t_m}{t_n}.
\]
Also compute
\[
L_*(\text{particle},n)=\frac{\hbar t_n}{m_{\text{particle}}c}
\]
and test clustering across a predeclared family.

\section{What the Synthesis Would Predict}
If a shared $L_*$ exists in a physical family, mass ratios should reflect closure-eigenvalue ratios.
If no stable $L_*$ appears, the zeta--mass link remains structural rather than predictive.
This is where the series becomes falsifiable.

\section{Non-claim boundary and falsification}
No claim is made here that known particle masses have already been derived. The framework is falsified in current form if only post hoc fits are possible and no stable constrained $L_*$ emerges.
