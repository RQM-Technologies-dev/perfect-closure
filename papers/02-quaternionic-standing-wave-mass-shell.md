## Abstract

This paper provides the geometric layer of the Perfect Closure thesis by expressing mass as stable quaternionic phase return. The central object is a unit quaternion
\[
q=\cos\phi+\mathbf u\sin\phi,
\]
with orientation axis \(\mathbf u^2=-1\). Free radiation corresponds to non-closing phase transport, while massive modes correspond to closed standing-wave phase patterns that repeatedly return to identity under conjugate evolution. This paper does not claim a microscopic derivation of all constants; it establishes the geometric language needed for the spectral shell bridge.

## 2. Quaternionic phase geometry

Represent phase orientation by
\[
q=\cos\phi+\mathbf u\sin\phi,
\]
where \(\mathbf u\) is a unit imaginary quaternion axis. The quaternionic conjugate is
\[
\bar q=\cos\phi-\mathbf u\sin\phi,
\]
and unit norm implies \(q\bar q=1\).

This geometry captures phase orientation and return without requiring a preferred complex plane. Different choices of \(\mathbf u\) correspond to different closure orientations.

## 3. Closure and standing-wave return

Let a phase mode evolve by repeated quaternionic action. Two qualitative regimes appear:

1. **Open/non-closing transport:** phase continues without self-return at a fixed closure cadence.
2. **Closed/standing mode:** conjugate paths repeatedly return to identity and preserve a bounded pattern.

The standing case is interpreted as mass-like because it admits a rest-frame resonance structure. This is the geometric precursor to the shell-number spectrum developed in Paper 3.

## 4. Mass-shell interpretation

The relativistic invariant
\[
P\bar P = m^2c^2
\]
can be read as a closure norm condition: a positive invariant corresponds to persistent closed structure, while null propagation corresponds to unresolved open transport.

This reinterpretation is conservative: it does not replace relativity, but changes explanatory emphasis from "material substrate" to "phase-closure state."
